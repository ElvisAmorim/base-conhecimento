# Portal do Conhecimento

## 👋 Bem-vindo, eu mesmo 😂

## Finalidade:

Esse portal de documentacao e conhecimento foi desenvolvido para centralizar todo o meu conhecimento como um segundo cerebro.

## Orientacao desse portal

* As guias são separadas por tipo de tecnologia

* Os tipos de tecnologia são:
    
    - **Aprendizado** - Anotacoes de cursos, aulas, livros e descobertas
    - **Trechos** - Codigos reutilizaveis
    - **Erros** - Problemas que eu resolvi
        - Template sugerido:

                ## Breve descricao
                ## Mensagem
                ## Causa
                ## Solucao
                ## Codigo corrigido
                
    - **Conceitos** - Explicacoes importantes
    - **Projetos** = Coisas que eu contrui
        - Exemplo:

                Analise de vendas
                Automacao relatorio
